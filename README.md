# Azimov Abdulahadxon Portfolio — Netlify Static Version

Bu versiya Netlify’da deploy xatosi bermasligi uchun Next.js’dan toza statik HTML/CSS/JS holatiga o‘tkazildi.

## Netlify sozlamalari
- Build command: `echo 'Static site - no build needed'`
- Publish directory: `.`
- Functions directory: `netlify/functions`

## Telegram sozlamasi
`netlify/functions/contact.js` faylida quyidagilarni almashtiring:
- `BOT_TOKENNI_SHU_YERGA_YOZING`
- `CHAT_IDNI_SHU_YERGA_YOZING`
