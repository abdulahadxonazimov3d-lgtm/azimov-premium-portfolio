# Azimov Premium Portfolio

## Ishga tushirish

```bash
npm install
npm run dev
```

## Netlify

Build command:

```bash
npm run build
```

Publish directory:

```text
.next
```

Telegram bot sozlamasi kod ichida yoziladi:

```text
app/api/contact/route.js
```

Ushbu qatorlarni o'zingizning bot ma'lumotlaringizga almashtiring:

```js
const TELEGRAM_BOT_TOKEN = 'BOT_TOKENNI_SHU_YERGA_YOZING';
const TELEGRAM_CHAT_ID = 'CHAT_IDNI_SHU_YERGA_YOZING';
```

GitHub linki kod ichida yozilgan joylar:

```text
app/page.js
app/components/Navbar.jsx
```
